# OlaForge - 智能安全沙箱系统架构设计

## 一、项目背景

本项目整合 SkillLite 和 CubeSandbox 的优点，创建一个更完美的 AI Agent 安全沙箱系统。

### 基础参考

| 项目 | 优点 | 缺点 |
|------|------|------|
| SkillLite | 自进化引擎、轻量级、零依赖 | 硬件隔离弱、集群能力弱 |
| CubeSandbox | 硬件级隔离、高性能、企业级 | 无智能化、部署复杂、运维要求高 |

---

## 九、安全分级设计

针对评审中提出的安全分级需求，设计四级安全模式：

### 9.1 安全等级对比

| 等级 | 名称 | 隔离方式 | 适用场景 | 安全评分 |
|------|------|----------|----------|----------|
| L0 | 快速模式 | 无隔离 | 可信代码、快速测试 | 0/20 |
| L1 | 轻量级 | namespace + caps | 个人开发、无第三方 | 12/20 |
| L2 | 标准级 | bwrap/seccomp | 一般使用、外部代码 | 16/20 |
| L3 | 增强级 | KVM MicroVM | 敏感场景、高安全 | 20/20 |

### 9.3 自动选择策略

```python
# 自动安全等级选择
def auto_security_level(code_source: str, has_network: bool) -> str:
    if is_trusted_code(code_source):
        return "L0"  # 快速模式
    elif has_network and not is_verified():
        return "L3"  # 增强级
    else:
        return "L2"  # 默认标准级
```

### 15.2 主配置文件格式

```yaml
# ~/.olaforge/config.yaml
version: "1.0"

# 模型配置
model:
  provider: "openai"              # openai/anthropic/local
  name: "claude-3-5-sonnet"       # 模型名称
  api_key: "${OLA_API_KEY}"       # 环境变量引用
  base_url: ""                    # 自定义 API 端点
  timeout: 60                     # 请求超时 (秒)
  max_retries: 3                  # 最大重试次数

# 安全配置
security:
  level: "L2"                     # L0/L1/L2/L3
  auto_level: true                # 自动选择安全等级
  network_allowed: false          # 是否允许网络
  file_write_allowed: false       # 是否允许写文件

# 执行配置
execution:
  timeout: 60                     # 执行超时 (秒)
  memory_limit: 4096              # 内存限制 (MB)
  cpu_limit: 2                    # CPU 核心数
  temp_dir: "/tmp/olaforge"      # 临时目录

# 进化配置
evolution:
  enabled: true                   # 是否启用进化
  auto_save: true                 # 自动保存
  feedback_threshold: 0.8         # 反馈阈值
  max_memory_items: 1000         # 最大记忆条目

# 日志配置
logging:
  level: "info"                   # debug/info/warn/error
  file: "~/.olaforge/logs/olaforge.log"
  max_size: 10                    # 单个日志文件大小 (MB)
  max_files: 5                    # 保留文件数

# 高级配置
advanced:
  cache_enabled: true             # 启用缓存
  preload_models: false           # 预加载模型
  parallel_execution: false       # 并行执行
  sandbox_engine: "bwrap"         # bwrap/firejail/none
```

result = client.run_skill(

for chunk in client.stream_chat("讲个故事"):
